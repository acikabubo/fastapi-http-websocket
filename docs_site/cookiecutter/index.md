# Cookiecutter Template

This project can be used as a Cookiecutter template to generate new projects.

## Contents

- [Overview](overview.md) - Template features and structure
- [Customization](customization.md) - Customizing the generated project
- [Examples](examples.md) - Example projects built with this template

## Quick Start

\`\`\`bash
# Install cookiecutter
pip install cookiecutter

# Generate new project
cookiecutter gh:yourusername/fastapi-http-websocket

# Follow the prompts to customize your project
\`\`\`

See [Overview](overview.md) for more information.
